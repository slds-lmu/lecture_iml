\aufgabe{Interactions}{
\label{ex:interaction_calculation}

Consider the following function: 
\begin{align*}
	f(\xv) = f(x_1, x_2) = 2 x_1 + 3 x_2 - x_1 |x_2|
\end{align*}
Mathematically check whether interactions are present.

% Idea: Sqrt-fct in x_1 only on [0, \infty], and then interaction with x_2
% Then consider distribution only on [-\infty, 0] on x_1  => No interactions

% Or task: Given a function, find 2 distributions such that interactions are present or not.

\newpage

Here, we consider the following definition of interactions: A twice differentiable function (or model) \(f\) depending on two features \(x_i\) and \(x_j\) (among others) exhibits an interaction between \(x_i\) and \(x_j\) if and only if

\[
\E \left[ \left( \frac{\partial^2 f(\xv)}{\partial x_i \partial x_j} \right)^2 \right] > 0 .
\]

}

\item \label{Ex:interaction_depending_on_distribution_a}
Given the function

$$
g(x_1, x_2) = 0.01e^{x_1^2} + \sin(x_2) \sqrt{x_1 \one_{(0, \infty)} (x_1)} - 1.5 x_2^3 ,
$$

where \(\one\) denotes an indicator function
$$
\one_{S}(x) = \left\{
\begin{aligned}
    1 \text{ iff } x \in S, \\
    0 \text{ iff } x \notin S,
\end{aligned}
\right. \hspace*{.2cm} \text{ and i.p. } \hspace*{.2cm} \one_{(a,b)}(x) = \left\{
\begin{aligned}
    &1 \text{ iff } a < x < b, \\
    &0 \text{ else,}
\end{aligned}
\right.
$$
find two distributions on \(\R^2\) such that the function \(g\) contains interactions w.r.t. one of them, but not the other.

\item
Again, consider the function \(f\) from exercise \ref{ex:interaction_calculation}, does it exhibit interactions w.r.t. each of the two distributions you found in \ref{Ex:interaction_depending_on_distribution_a})? Can you find any distribution that would change your result?

\end{enumerate}
